## 新特性

- 增加了铜制装饰品

### 铜制装饰品

- 增加了铜火把
  - 一个使用铜粒、煤炭与木棍合成的绿色的火把变体
  - 和火把的作用一样


- 增加了铜栏杆
  - 和现存的铁栏杆的作用一样
  - 会随时间氧化


- 增加了铜链
  - 和现存的锁链（现在重命名成铁链了）的作用一样
  - 会随时间氧化


- 增加了铜灯笼
  - 与普通灯笼合成配方相似但使用铜以及铜火把
  - 灯笼会自己氧化，随着时间外观也在变化
  - 和现存的灯笼的作用一样

## 数据包版本 83.0

### 实体数据

#### `copper_golem`

- `weather_state` 字段现在期望一个字符串 id 而不再是整数 id
  - 0 -> `unaffected`
  - 1 -> `exposed`
  - 2 -> `weathered`
  - 3 -> `oxidized`

### 文本渲染

#### `object` 文本组件

- 添加了新的文本组件 `object`
- 这个组件会从精灵图纹理图集以字符显示单个精灵图
- 精灵图一直会以 8x8 像素的矩形渲染
- 格式:
  - `atlas` - 纹理图集的命名空间 ID，默认值: `minecraft:blocks`
  - `sprite` - 在纹理图集中的精灵图命名空间 ID，例如： `item/porkchop`

### 战利品表

- 增加了 `harvest/sweet_berry_bush` 战利品表，用于交互收获有浆果的浆果丛的掉落

### 标签

#### 方块标签

- 增加了 `#chains` - 所有锁链方块
- 增加了 `#lanterns` - 所有灯笼方块
- 增加了 `#bars` - 所有金属栏杆方块

#### 物品标签

- 增加了 `#chains` - 所有锁链方块物品
- 增加了 `#lanterns` - 所有灯笼方块物品
- 增加了 `#bars` - 所有金属栏杆方块物品
