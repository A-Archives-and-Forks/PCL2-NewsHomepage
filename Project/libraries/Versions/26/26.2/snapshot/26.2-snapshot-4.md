## 数据包版本 103.0

### 属性

#### 增加了 `minecraft:nameplate_distance`

- 控制实体 nameplate 在几格之内可见
- 若实体不可见， nameplate 也不可见
- 接受 `0.0` 到 `512.0` 之间的值
- 默认值: `64.0`

#### 增加了 `minecraft:below_name_distance`

- 控制实体 `below_name` 记分板显示在几格之内可见
- 若实体不可见， nameplate 也不可见
- 接受 `0.0` 到 `512.0` 之间的值
- 默认值: `10.0`

### 遥测

#### 新必要事件

- `graphics_capabilities`
  - 一个新的事件，在启动时推送，用于告知我们用于进行游戏的图形设备的功能
  - 增加了新的属性: `backend_name`
    - 这项值为 `Vulkan` 或 `OpenGL`，这会帮助我们了解玩家正在使用哪个图形 API 进行游戏
  - 增加了新的属性: `backend_failure_reason`
    - 这是一个错误代码列表，例如 `vulkan_device_version_too_low`，这将帮助我们识别对 Vulkan 需求的问题
  - 增加了新的属性: `backend_failure_message`
    - 这是一个简短且模糊的信息，例如 `Failed to find the GLFW platform surface extensions`，如果玩家正在使用 Vulkan 时发生问题，这将帮助我们缩小问题范围
  - 增加了新的属性: `backend_failure_missing_capabilities`
    - 这是一个缺失 Vulkan 的功能的列表，例如 `VULKAN_CORE_1_2, VK_KHR_dynamic_rendering`，这将帮助我们了解玩家使用 Vulkan 需要用到哪些功能
