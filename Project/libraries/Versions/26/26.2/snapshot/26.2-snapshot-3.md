## 新特性

### Sulfur Spike

- 一个可以自然生成于硫黄洞穴生物群系内的硫黄上的新的方块
- 放置在天花板上时会形成钟乳石，放置在地面上时会形成石笋
- 可以相接，组合成更长的钟乳石和石笋
- 当尖端互相紧挨时，钟乳石和石笋会合并为一体，除非在放置时按 Shift
- 石笋下无方块支撑时会被破坏
- 钟乳石上无方块连接时会落下
- 被落下的钟乳石砸到会受伤，而且它们非常锋利！
- 向其投掷三叉戟可以破环

### 谓词

#### 实体谓词

实体谓词格式已从有着多个可选字段的结构改成一个与数据组件相似的映射。

例如，之前 `effects` 是一个可选字段：

```
{
    "effects": {...}
}
```
在本版本中，其变为类似组件的子谓词条目：

```
{
    "minecraft:effects": {...}
}
```

主要功能更改如下：

- 所有在实体谓词里的键现在为 ID
  - 因为 ID 可以不带命名空间（默认为 `minecraft` 命名空间），现存的字段依旧生效
  - 例外: 字段 `type` 已被重命名为 `minecraft:entity_type`
- 无法识别的子谓词组件现在会被拒绝（此前未知的字段会被忽略）

现有的类型特定子谓词已被重命名并移动到顶层。

- `minecraft:lightning` 重命名为 `minecraft:type_specific/lightning`
- `minecraft:fishing_hook` 重命名为 `minecraft:type_specific/fishing_hook`
- `minecraft:player` 重命名为 `minecraft:type_specific/player`
- `minecraft:raider` 重命名为 `minecraft:type_specific/raider`
- `minecraft:sheep` 重命名为 `minecraft:type_specific/sheep`
- `minecraft:slime` 重命名为 `minecraft:cube_mob` 因为其包含史莱姆、岩浆怪和新的 Sulfur Cube

例如：

```
{
  "type_specific": {
    "type": "minecraft:player",
    "looking_at": {
      "type": "minecraft:ender_dragon"
    }
  }
}
```
变为

```
{
  "minecraft:type_specific/player": {
    "looking_at": {
      "minecraft:type": "minecraft:ender_dragon"
    }
  }
}
```
##### 增加了 `minecraft:entity_tags` 实体子谓词

这个谓词匹配实体标签 (例如，在 `/tag` 命令中设置的标签)。

字段：

- `any_of` - 可选字符串列表, 选中的实体必须有列表中任一标签
- `all_of` - 可选字符串列表, 选中的实体必须有列表中全部标签
- `none_of` - 可选字符串列表, 选中的实体必须没有列表中的所有标签

### 世界生成

#### 主世界地物

- 调整了地物类型 `lake`
  - 增加了字段 `can_place_feature` - 方块谓词, 描述可以放置该地物的方块
  - 增加了字段 `can_replace_with_air_or_fluid` - 方块谓词, 描述地物可以替换成空气或在 `fluid` 中提供的方块
  - 增加了字段 `can_replace_with_barrier` - 方块谓词, 描述该地物可以用提供的 `barrier` 替换哪些方块

### 标签

#### 方块标签

- 增加了 `#speleothems` ，包含类似于石锥的方块，如小型滴水石锥和 Sulfur Spike
