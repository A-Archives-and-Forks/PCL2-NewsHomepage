> [!INFO]
> 目前登录API已恢复正常

Mojang API 于 5月28日23时许频繁返回 503、504，无法正常使用。

以下功能可能受到影响：

* **正版登录**
* 服务器正版验证
* 其它操作

部分 API 仍未恢复，**但官方启动器仍能启动**，若有需要可临时使用官方启动器启动游戏。

* 0时6分更新：MOJANG发布说正在处理，但无法给出恢复预计时间
* 0时22分更新：目前已能够正常登录
* 0时52分更新：官方已确认修复完成

> [!IMPORTANT]
> 由于 MOJANG 服务器造成的无法登录与启动器无关，当前使用第三方启动器进行正版登录都会失败，请不要就此问题反馈给启动器开发者
